# Elijeweb - Your AI Sales Assistant for Meta (Full-Stack Angular/NestJS)This project is a full-stack application built with Angular (frontend), NestJS (backend), Prisma (ORM), and SQLite (database), styled with Tailwind CSS. It simulates an AI sales assistant dashboard for social media businesses.**Features:**- User Authentication (Register, Login, JWT-based)- Dashboard with Sales Overview, Recent Orders, Stock Levels, and Key Metrics- Product Management (CRUD operations - *backend only, UI not fully implemented in this version*)- Order Management (CRUD operations - *backend only, UI not fully implemented in this version*)- Dark Mode Toggle- Responsive Design## Getting StartedFollow these steps to set up and run the project on your local machine.### Prerequisites- Node.js (LTS version recommended)- npm (comes with Node.js)- Git### Installation1.  **Clone the repository:**    ```bash    git clone https://github.com/your-username/elijeweb.git    cd elijeweb    ```2.  **Create environment file:**    Create a `.env` file in the root directory of the project based on `.env.example`.    ```dotenv    # .env    DATABASE_URL="file:./dev.db"    JWT_SECRET="your_super_secret_jwt_key_here" # Replace with a strong, random secret    ```3.  **Install dependencies:**    This will install dependencies for both the Angular frontend and the NestJS backend.    ```bash    npm install    ```4.  **Set up the database:**    Run Prisma migrations to create the SQLite database file (`dev.db`) and tables.    ```bash    npx prisma migrate dev --name init    ```    *Note: If you make changes to `prisma/schema.prisma`, you'll need to run `npx prisma migrate dev` again.*5.  **Start the application:**    This command uses `concurrently` to start both the NestJS backend and the Angular frontend simultaneously.    ```bash    npm run start    ```    - The NestJS backend will run on `http://localhost:3000` (watching for changes).    - The Angular frontend will run on `http://localhost:4200` (and open in your browser).## Project Structure```plaintextelijeweb/├── node_modules/├── dist/ #
